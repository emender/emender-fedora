##YAML format
```
- word or phrase:
    desc: "Here should be description of word or phrase. There are some rules: It is necessary to remove all double quotes and use apostrophe instead of them. Then if the string contains colon it'll be necessary to enclose the whole string with double quotes."
    type: This deteremines whether described phrase is topic (i.e. abbreviations) or word which is used in the text (i.e. can). For topic use key word 'topic' for words 'word'.
    class: Part of speech (or leave it epmty when describing topic).
    use_it: yes, no or avoid
    refers: Here it is posible to list phrases which are related to this phrase.
```
